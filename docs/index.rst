=======
ieee754
=======

Arbitrary-precision IEEE-754 Arithmetic in Python.

The current version is |release|.

Source Code
===========

The project is hosted on `GitHub <https://github.com/kyuupichan/ieee754/>`_.  and uses
Azure Pipelines for Continuous Integration.

Please submit an issue on the `bug tracker
<https://github.com/kyuupichan/ieee754/issues>`_ if you have found a
bug or have a suggestion to improve the library.
